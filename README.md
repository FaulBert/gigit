  # Gigit

  ##### gigit user/repo

</span>

---

Git clone made easy!
Cloning a GitHub repository is easy
and error-free (only one error)

It does work, but only for cloning.
Use [degit](https://github.com/Rich-Harris/degit) for maximum minimalism

### Usage

Clone repo

```sh
gigit <user>/<repo>
```


Clone repo with --depth=1

```sh
gigit -1 <user>/<repo>
```
